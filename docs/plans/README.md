# Plans

Implementation plans for non-trivial work. Drafted in conversation, then converted to a markdown doc here so multi-PR work has a stable reference.

```
docs/plans/
├── active/    # in-progress — at least one action item still open
└── archive/   # all action items shipped, kept for historical context
```

Move a doc from `active/` to `archive/` (via `git mv`) once every action item it tracks is merged. Don't delete — old plans document why decisions were made and what trade-offs were considered.
